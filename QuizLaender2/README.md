Ein Quizspiel zum Lernen von Länderflaggen, entwickelt mit C# und WPF nach dem MVVM-Pattern.

Features:
---------

Flaggen-Quiz mit mehreren Antwortmöglichkeiten

Punktesystem (Score)

Timer pro Frage

Leben-System

Zufällige Länder aus JSON-Datei

Moderne WPF Oberfläche

Technologien:
-------------

C#

WPF

MVVM Architektur

JSON Datenquelle

DispatcherTimer

Projektstruktur:
----------------
QuizLaender2
│
├── Models
│   └── Country.cs
│
├── ViewModels
│   └── QuizViewModel.cs
│
├── Views
│   └── MainWindow.xaml
│
└── Data
    └── Countries.json

Projekt starten:
---------------

Repository klonen: git clone https://github.com/kaliffo/QuizLaender2.git

Projekt in Visual Studio öffnen.

Starten mit F5
